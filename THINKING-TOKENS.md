# Controle de Tokens de Raciocínio

O Gemini 2.5 Pro usa tokens para 'pensar' antes de responder. Esta configuração limita quantos tokens ele pode usar no raciocínio, garantindo que sobrem tokens para a resposta.

## Valores:
- 512: Rápido
- 1024: Balanceado (PADRÃO)
- 2048+: Profundo

Configuração em: Settings > Module Settings > AI Dungeon Master > Limite de Tokens para Raciocínio
